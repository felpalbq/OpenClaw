# COGNITIVE_FRAMEWORK.md — Framework de Raciocínio Criativo

> Este documento não é uma skill. É o sistema de raciocínio criativo do sistema.
> Ele define COMO pensar, não o que fazer.
> Aplicável quando o estado indica que estratégia profunda é necessária (condition: deep_strategy).

---

## Princípio Central

Conteúdo relevante não é sobre o tema — é sobre a CONEXÃO entre o que está acontecendo e o que o cliente representa. Conexão superficial gera conteúdo forçado. Conexão genuína gera conteúdo que o público reconhece sem precisar de explicação.

---

## As 4 Camadas de Raciocínio

### Camada 1 — Mapeamento de Domínios

Antes de conectar, mapear separadamente cada domínio presente no input.

Para cada domínio, identificar suas dimensões:
- Dimensão emocional: o que as pessoas SENTEM sobre isso
- Dimensão comportamental: como as pessoas AGEM em relação a isso
- Dimensão situacional: em que CONTEXTO isso acontece
- Dimensão relacional: como isso afeta RELAÇÕES entre pessoas ou entre pessoas e coisas

Exemplo — BBB como domínio:
- Emocional: torcida, rivalidade, empatia, julgamento, identificação
- Comportamental: confinamento, adaptação, performance sob pressão, convivência forçada
- Situacional: isolamento do mundo externo, rotina monitorada, exposição constante
- Relacional: alianças, conflitos, afeto em ambiente de tensão

Exemplo — Veterinária como domínio:
- Emocional: amor incondicional, preocupação, culpa, alívio
- Comportamental: cuidado diário, rotina do pet, resposta do animal ao humano
- Situacional: apartamento pequeno, ausência do tutor, ambiente restrito
- Relacional: vínculo humano-animal, dependência mútua, comunicação não-verbal

### Camada 2 — Identificação de Ressonância

Cruzar as dimensões dos dois domínios buscando onde se encontram de forma NÃO ÓBVIA.

Conexão óbvia (evitar): BBB + veterinária = "seu pet também merece um líder" — transposição literal, não ressonância.

Conexão genuína (buscar): confinamento do BBB ressoa com comportamento de pets em apartamento pequeno — ambos lidam com espaço restrito, rotina monitorada, dependência emocional de quem está próximo. Isso é ressonância comportamental.

Pergunta de teste: "Se eu mostrar essa conexão para alguém sem explicar, ela vai entender e se reconhecer?" Se sim → ressonância genuína. Se precisar de explicação → conexão forçada.

### Camada 3 — Teste de Autenticidade

Antes de desenvolver um ângulo, verificar:

1. A conexão tem uma verdade emocional real? (não é apenas analogia intelectual)
2. O público do cliente VIVE essa ressonância no dia a dia?
3. A conexão serve o objetivo do cliente ou apenas é criativa por ser criativa?
4. O timing é real — essa ressonância faz sentido AGORA?

Se qualquer resposta for não → descartar e voltar à Camada 2.

### Camada 4 — Tradução para Formato

Com a ressonância autêntica identificada, definir como ela se manifesta melhor:

- Narrativa: quando a conexão conta uma história
- Contraste: quando a conexão revela uma inversão de expectativa
- Analogia direta: quando a conexão ilumina algo novo
- Pergunta retórica: quando a conexão ativa reconhecimento imediato
- Dado + contexto: quando a conexão é reforçada por fato relevante

O formato escolhido deve amplificar a ressonância, não apenas contê-la.

---

## O que NÃO é conexão contextual

- Usar tema viral como cenário decorativo
- Forçar analogia que precisa de explicação para fazer sentido
- Aproveitar trend sem relevância real para o nicho
- Criar conexão intelectualmente interessante mas emocionalmente fria

---

## Quando usar este framework

Quando o estado indica que estratégia profunda é necessária — conteúdo com condition `deep_strategy`.

Para inputs diretos e literais, usar `content_strategy` normalmente sem este framework.

Se nenhuma ressonância autêntica é encontrada após 3 tentativas, reportar e sugerir voltar a `content_strategy`.
