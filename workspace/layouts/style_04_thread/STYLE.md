# STYLE — THREAD

## Tipo de uso

Usar quando:
- conteúdo é textual
- narrativa contínua
- simulação de conteúdo nativo

Evitar quando:
- precisa de impacto visual forte

---

## Características

- densidade: alta
- estrutura: contínua
- uso de imagem: baixo
- hierarquia: simples

---

## Comportamento

- ritmo: contínuo
- leitura: fluida

---

## Output esperado

- sensação de conteúdo orgânico
- baixa fricção
